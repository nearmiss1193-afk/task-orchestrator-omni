## THE LEAD GENERATION HIERARCHY

### Inbound (Leads Come to You)

1. **SEO/Organic** - Free, slow, compounds over time
2. **Content Marketing** - Blog, YouTube, podcasts
3. **Referrals** - Highest trust, highest close rate
4. **Reviews/Reputation** - Google reviews = local SEO

### Outbound (You Go to Leads)

1. **Cold Email** - Scalable, low cost
2. **Cold Calling** - Direct, immediate feedback
3. **LinkedIn Outreach** - B2B focused
4. **Door Knocking** - Service industry staple

### Paid (You Pay for Leads)

1. **Google Ads** - Intent-based, expensive but converts
2. **Facebook/Meta** - Cheap CPM, good for awareness
3. **Local Services Ads (LSA)** - Pay per lead, Google verified
4. **Lead Aggregators** - Angi, Thumbtack, HomeAdvisor

## ðŸŽ¯ CHANNEL-BY-CHANNEL DEEP DIVES

### Google Ads for Local Services

```
Best practices:
- Use location extensions
- Bid on "[service] near me" keywords
- Negative keyword: "jobs", "DIY", "free"
- Call extensions are CRITICAL
- Landing page must match ad intent
```

**Benchmarks:**

- CPC: $15-50 for home services
- Conversion rate: 5-15%
- Cost per lead: $50-200

### Facebook/Instagram Ads

```
Best practices:
- Video > Image > Carousel
- Lead forms inside Facebook (don't send to website)
- Retarget website visitors
- Lookalike audiences from customer lists
- Testimonial content performs best
```

**Benchmarks:**

- CPM: $10-25
- CPC: $1-5
- Cost per lead: $20-80

### Cold Email Best Practices

```
1. Personalize the first line (reference their business)
2. Keep it under 100 words
3. One clear CTA
4. Send 3-5 follow-ups
5. Use tools: Instantly.ai, Apollo.io, Smartlead

Subject lines that work:
- "Quick question for [Company Name]"
- "[Mutual connection] suggested I reach out"
- "Idea for [specific problem they have]"
```

**Benchmarks:**

- Open rate: 40-60% (cold)
- Reply rate: 2-8%
- Meeting booked rate: 0.5-2%

### LinkedIn Outreach

```
Best practices:
1. Optimize profile first (professional photo, clear headline)
2. Connect with personalized note
3. Wait 24 hours, send value message
4. Never pitch in first message
5. Use Sales Navigator for targeting
```

## ðŸ”„ THE LEAD FOLLOW-UP MACHINE

### Speed-to-Lead Data

- **Calling within 5 minutes:** 100x more likely to connect
- **After 30 minutes:** Lead is basically dead
- **78% of deals** go to first responder

### The 7-Touch Follow-Up Sequence

| Day | Channel | Message |
|-----|---------|---------|
| 0 | Call + SMS | Immediate response |
| 0 | Email | Confirmation + info |
| 1 | SMS | "Did you get my message?" |
| 2 | Call | Voicemail |
| 3 | Email | Value add (tips, guide) |
| 5 | SMS | "Last chance" urgency |
| 7 | Email | Break-up email |

### The Break-Up Email (Magic)

```
Subject: Should I close your file?

Hi [Name],

I've tried reaching you a few times without luck. 
I don't want to be a pest, so I'm going to assume 
the timing isn't right.

If things change, you have my number: [phone]

Best,
[Your Name]
```

*This email gets the highest response rate*

### Website Must-Haves

1. Phone number HUGE at top
2. Click-to-call on mobile
3. Form above the fold
4. Trust badges (BBB, licenses, reviews)
5. Clear service areas listed
6. Before/after photos
7. Video testimonials
8. Live chat or chatbot

### Landing Page Formula

```
Headline: Clear benefit or solution to pain
Subheadline: How you do it differently
Social proof: Reviews, logos, certifications
Form: Name, phone, email, service needed
CTA: "Get Your Free Quote" (not "Submit")
```

## ðŸ“Š TRACKING & ATTRIBUTION

### What to Track

1. **Source** - Where did they come from?
2. **Cost** - How much did we spend?
3. **Leads** - How many?
4. **Appointments** - How many scheduled?
5. **Shows** - How many showed up?
6. **Closes** - How many bought?
7. **Revenue** - How much made?

### Tools for Tracking

- **CallRail** - Call tracking with dynamic numbers
- **Google Analytics** - Website behavior
- **GHL** - Full funnel tracking
- **Hyros** - Advanced attribution
- **Triple Whale** - E-commerce attribution

### The Formula

```
Revenue - Ad Spend = Profit
Profit / Ad Spend = ROAS (Want 3x+)
Revenue / Leads = Value per Lead
Cost / Leads = Cost per Lead
Value per Lead > Cost per Lead = WIN
```
