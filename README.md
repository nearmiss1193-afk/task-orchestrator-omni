# Railway Deployment

This is the Empire Unified cloud worker for Railway deployment.

## Environment Variables Required

- APOLLO_API_KEY
- SUPABASE_URL
- SUPABASE_KEY
- VAPI_PRIVATE_KEY
- GHL_API_TOKEN
- GHL_LOCATION_ID
- GEMINI_API_KEY

## Features

- Prospecting every 2 hours (Apollo API)
- Email outreach via GHL webhooks
- SMS outreach via GHL webhooks
- Vapi call triggers
- Inbound SMS response polling
